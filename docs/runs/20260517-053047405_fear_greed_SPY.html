fear_greed on SPY

Contrarian timing using CNN's Fear & Greed Index — a daily 0–100 sentiment score blending price momentum, volatility, breadth, put/call ratios, and safe-haven demand. We buy the ticker when sentiment falls below buy_below (default 25, 'extreme fear') and exit to cash when it climbs above exit_above (default 75, 'extreme greed'). The premise is that crowd extremes mark turning points. Two data sources are available (picked via --fg-source): 'cnn' uses the live CNN endpoint (~5.5 years of history, includes component sub-indices), 'whit3rabbit' uses a community CSV mirror reaching back to 2011-01-03 (headline score only).
